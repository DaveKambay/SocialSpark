SocialSpark

SocialSpark is an Android app that helps you stay socially connected by giving quick suggestions based on the time of day.

Features
Enter a time of day to get a social suggestion.
Suggestions include texts, calls, or other small social gestures.
Reset button to clear input and start over.
Logs user activity for testing and debugging.
How It Works
Type a time of day (e.g., morning, afternoon, dinner) into the input box.
Click "Find a Spark".
The app displays a suggested social activity.
Click "Reset" to clear the screen.

Supported times:

Morning
Mid-morning
Afternoon
Afternoon snack time
Dinner
Night / After dinner
